# 로그인 오류 해결 (Invalid username or password)

## 1. 확인할 것

- **사용자 ID**: `admin` (소문자 권장)
- **비밀번호**: `admin123`
- **공백**: ID/비밀번호 앞뒤에 공백이 없어야 합니다. 다시 입력해 보세요.

이제 **사용자 ID는 대소문자 구분 없이** 동작합니다. `Admin`, `ADMIN`으로 입력해도 됩니다.

---

## 2. 그래도 안 될 때

### 방법 A: 백엔드 재시작

1. 백엔드 서버를 **완전히 종료**합니다.
2. 아래처럼 다시 실행합니다.

```bash
cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

3. 기동 시 시드 데이터(테스트 사용자)가 다시 적용됩니다.  
   `admin` / `admin123` 로 다시 로그인해 보세요.

### 방법 B: DB 초기화 후 재시작

기존 DB 파일이 꼬였을 수 있으므로, **DB를 지우고** 백엔드를 다시 띄웁니다.

1. 백엔드를 종료합니다.
2. **backend** 폴더 안의 `jscorp_hr.db` 파일을 **삭제**합니다.
3. 백엔드를 다시 실행합니다.

```bash
cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

4. 새 DB가 생성되고 테스트 사용자가 다시 만들어집니다.  
   `admin` / `admin123` 로 로그인합니다.

## 4. API 주소 설정 (로컬)

- 백엔드를 다른 포트로 띄웠다면 그 주소로 바꿉니다 (예: http://localhost:8001).
